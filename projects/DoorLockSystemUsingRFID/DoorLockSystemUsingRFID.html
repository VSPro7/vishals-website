Door Lock System using RFID

Introduction

In this project we are going to make a door lock system using an arduino uno and an ESP32, where the Arduino is going to be placed outside the door and whenever a person scans an rfid card that is valid the HC-05 Bluetooth Module sends a message via bluetooth to the ESP32 that is placed inside the door and when the ESP32 recieves a message it uses a servo motor to open the door.

Hardware Required:-

  1. Arduino Uno
  2. ESP32 DEV KIT V1
  3. HC-05 Bluetooth Module
  4. Jumper Wires
  5. LCD Display 16X2
  6. Potentiometer
  7. RC522 RFID Reader
  8. Servo Motor
  9. Ultrasonic Sensor HC-SR04
  10. Breadboard (Not Necessary)

Software Required:-

  1. Arduino IDE

Arduino IDE Libraries:-

  1. BluetoothSerial
  2. ESP32Servo
  3. LiquidCrystal
  4. MFRC522
  5. SoftwareSerial

Bluetooth Serial Communication Setup

To setup a bluetooth connection between the HC-05 Bluetooth Module and the inbuilt Bluetooth Module of the ESP32 we first need to find the bluetooth address of the ESP32 and to do that you need to upload the code given in the "Arduino Projects\DoorLockSystemUsingRFID_Card\ESP32\ESP32BluetoothAddressFinder" folder or used the code given below and open it through the "Arduino IDE" software. After the code is uploaded to the ESP32 you have to open the serial monitor and press EN button to reset the board. Then some text will appear on the serial monitor, the last line of this text will be the bluetooth address of the ESP32. It will look something like this "X0:00:X0:XX:0X:X0".

Now that we have found the bluetooth address of the ESP32 we need to tell the HC-05 bluetooth module to connect to this address and to do this we used something called the AT commands. To use these AT commands we first need to upload a blank sketch to the arduino board and build the circuit given in the "Arduino Projects\DoorLockSystemUsingRFID_Card\ArduinoUno\HC-05 Bluetooth ModuleHC-05 - Circuit Diagram". After this unplug the arduino from the computer. Now press and hold the button on the HC-05 Bluetooth Module and plug the arduino back into the computer and release the button. Now the led on the bluetooth module will blink at intervals of two seconds, this shows that the module has entered AT command mode.

Now that AT command mode is turned on open the serial monitor on the arduino IDE software and type "AT" without the quotes and hit enter. If you get an "OK" you have entered AT command mode successfully or if you don't you can try again from paragraph two of "Bluetooth Serial Communication Setup" or you can find my email at the end of this document.

Now you have to tell the Bluetooth module to connect to the given Bluetooth address using AT commands. So, we will use an AT command called "AT+BIND". Now for example, if your bluetooth address is "X0:00:X0:XX:0X:X0" remove the first, fourth and fifth colon and it will look something like "X000:X0:XX0XX0". Now replace all the colons with commas and make every thing lowercase to make it look like "x000,x0,xx0xx0". So, if your bluetooth address is "12:34:56:AB:CD:EF" then it will look like "1234,56,abcdef". Then you will type "AT+BIND=(your bluetooth address)", for example "AT+BIND=1234,56,abcdef" in the serial monitor. If you get an "OK" you have paired your ESP32 and HC-05 Bluetooth Module successfully or if you don't you can try again from paragraph two of "Bluetooth Serial Communication Setup" or you can find my email at the end of this document.

Now that we have paired the ESP32 and the HC-05 Bluetooth Module together, it is time to start with the actual project.
